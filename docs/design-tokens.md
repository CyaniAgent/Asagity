# Asagity Design Tokens (v2026 Sync)

本文档定义了 Asagity 平台的核心设计令牌（Design Tokens），旨在确保 **SSS 级 (2D/3D 混合)** 的视觉一致性。

## 🎨 色彩系统 (Color Palette)

核心品牌色为 **初音绿 (#39C5BB)**，在 Tailwind 配置中映射至 `cyan` 系列。

| Token | Hex/Value | 用途 |
| :--- | :--- | :--- |
| `primary` | `#39C5BB` | 核心高亮、按钮、光效 (Miku Green)。 |
| `accent` | `#22D3EE` | 辅助青色，用于外发光投影。 |
| `bg-base` | `#121212` | 深色模式基础底色。 |
| `bg-surface` | `rgba(255, 255, 255, 0.05)` | 玻璃态卡片底色。 |
| `text-high` | `rgba(255, 255, 255, 0.95)` | 高对比度文字。 |
| `text-dim` | `rgba(255, 255, 255, 0.4)` | 辅助说明文字。 |

### 🚨 动态亮度感应 (Luminance Sensing)
为了确保跨组件的文字可读性，系统执行以下自动配色逻辑：
- **阈值公式**：`0.2126*R + 0.7152*G + 0.0722*B`
- **判定条件**：当背景亮度 **> 140** 时，文字强制切换为 **Black (`#000000`)**，否则保持 **White (`#FFFFFF`)**。

## 💎 图标体系 (Iconography: MD3)

项目已全面从 Lucide 迁移至 **Material Design 3 (Material Symbols)**。

- **规范**：优先使用 `Rounded` 变体。
- **调用方式**：`i-material-symbols-{icon-name}-rounded`。
- **关键映射示例**：
    - 消息：`chat-bubble`
    - 回复：`subdirectory-arrow-right`
    - 位置：`location-on`
    - 循环：`repeat`

## ✨ 沉浸式视效 (Immersive Effects)

### 歌词焦点模糊 (Spotify Focus-Blur)
用于打造 SSS 级沉浸式听歌体验。
- **活动行 (Active)**：`opacity-100`, `scale-105`, `font-black`, `drop-shadow-2xl`。
- **非活动行 (Inactive)**：`opacity-20`, `blur-[2.5px]`, `scale-95`。
- **边缘遮罩**：`mask-image: linear-gradient(to bottom, transparent, black 15%, black 85%, transparent)`。

### 玻璃态参数 (Glassmorphism)
- **模糊等级**：`blur-md` (普通菜单) / `blur-3xl` (沉浸式浮窗)。
- **容器圆角**：`30px` (Container) / `full` (Button)。

## 📐 布局比例 (Layout Ratios)

### 设置页面 (Settings/Profile)
- **侧边栏/详情通栏**：设置中心导航宽度建议在 `w-settings-nav` (约 240px)。
- **个人档案编辑**：采用 **48 : 剩余空间** 的左右排版比例，确保头像 (`w-48`) 与横幅不重叠。
